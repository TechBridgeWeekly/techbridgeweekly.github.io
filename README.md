## Format（文章格式）
- 文章內容請遵守[中文文案排版指北](https://github.com/sparanoid/chinese-copywriting-guidelines)
- 若為翻譯文章，用詞請先搜尋相關用法，或可以參考這個[網站](http://jjhou.boolan.com/terms.htm)

---
title: Caffe & GoogLeNet，怎麼幫助機器人更好地辨識物體
